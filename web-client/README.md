# DentaLogic Web Client

This is the separate browser-first version of DentaLogic, built with React and Vite instead of Flutter.

## Run Locally

From this folder:

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.

## Build For Hosting

```bash
npm run build
```

The deployable website will be generated in:

```text
web-client/dist
```

## Defense Note

This web client is intentionally separate from the Flutter app. It uses a standard frontend web stack, responsive CSS, browser-first routing state, accessible controls, and a modern glassmorphism dashboard design.
